# Rodem Studio Web v0.1

휴대폰과 PC에서 사용할 수 있는 발매트 자동 배경 제거 웹앱입니다.

## 가장 쉬운 공개 방법: Streamlit Community Cloud

### 1. GitHub 계정 만들기
GitHub에 가입합니다.

### 2. 새 저장소 만들기
새 repository를 만들고 이 압축파일 안의 아래 파일들을 업로드합니다.

- `app.py`
- `requirements.txt`

### 3. Streamlit Community Cloud에서 배포
Streamlit Community Cloud에 GitHub 계정으로 로그인합니다.

- 저장소 선택
- Main file path: `app.py`
- Deploy 클릭

배포가 끝나면 휴대폰에서도 접속할 수 있는 인터넷 주소가 생성됩니다.

## 현재 기능

- 휴대폰 사진 선택
- 발매트 자동 배경 제거
- 투명 배경 미리보기
- 투명 PNG 저장
- 배경 제거 강도 조절
